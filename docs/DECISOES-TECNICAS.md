# Decisões técnicas

## Java e XML

O aplicativo utiliza Java e layouts XML para manter compatibilidade com o ecossistema Android tradicional e permitir controle explícito das telas e dos componentes.

## Funcionamento local

A primeira versão foi preparada para funcionar localmente. Isso permite validar a experiência principal antes da migração para uma infraestrutura remota definitiva.

## Evolução para API

As responsabilidades foram organizadas para permitir a substituição gradual do armazenamento local por uma API autenticada, sem exigir a reconstrução completa das telas.

## Privacidade

O portfólio utiliza somente capturas com dados fictícios. Credenciais, endpoints privados, regras de autorização e detalhes de persistência não fazem parte deste repositório.

## Identidade visual

O MeuReal possui identidade própria baseada em verde-petróleo, suporte a temas claro e escuro e componentes consistentes para navegação, cartões e ações financeiras.
